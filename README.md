# Cache

A PSR-16 simple cache implementation.

## System Requirements

You need `PHP => 8.1` but the latest stable version of PHP is recommended.

This package is continuously tested on the following PHP versions:

- PHP 8.1
- PHP 8.2
- PHP 8.3
- PHP 8.4
- PHP 8.5

## Installation

It's recommended that you use [Composer](https://getcomposer.org/) to install the cache repository.

```console
composer require zaphyr-org/cache
```
